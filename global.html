FreshSenseButton(onClick, loading, phase, total) → {JSX.Element}

Description:
  • Renders a button that initiates image analysis and displays loading states.
Source:
  • components/ui/features/vision/fresh-sense/FreshSenseButton.jsx, line 55
Example
```jsx
import React, { useState } from "react";
import { FreshSenseButton } from "@/components/ui/features/vision/fresh-sense/FreshSenseButton";

function ExampleUsage() {
  const [loading, setLoading] = useState({ vision: false, text: false });
  const [phase, setPhase] = useState(1);
  const totalPhases = 5;

  const handleAnalyze = () => {
    setLoading({ vision: true, text: false });
    // Simulate analysis steps...
    setTimeout(() => {
      setLoading({ vision: false, text: true });
      setTimeout(() => {
        setLoading({ vision: false, text: false });
        setPhase((prev) => prev + 1);
      }, 1000);
    }, 2000);
  };

  return (
    <FreshSenseButton
      onClick={handleAnalyze}
      loading={loading}
      phase={phase}
      total={totalPhases}
    />
  );
}
```
Parameters:
Name Type Description
onClick function Callback function invoked when the button is clicked. (required)
loading Object Object representing loading states: `vision` for vision analysis and `text` for final response generation. (required)
phase number Current analysis phase index, used to display progress. (required)
total number Total number of analysis phases. (required)
Returns:
A styled button that shows a loader and appropriate label text based on loading state.
Type
JSX.Element
